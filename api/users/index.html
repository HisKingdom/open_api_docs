账户管理

查看access_token对应用户的相关信息

通过access token查看对应用户的相关信息。

调用请求

HTTP请求方式: GET
https://api.convertlab.com/v1/users/current?access_token={access_token}

参数说明

  • {access_token}用从“获取身份权限”节拿到的access_token替换

返回说明

{
  "name": "Ha...",
  "loginName": "ha....f.com",
  "email": "han....@e.x.com",
  "tid": 11....47,
  "tname": "XXXXX有限公司"
}

对应信息模型

参数名 数据类型 说明
name string 用户名
loginName string 登录名
email string 邮箱
tid int tid
tname string 公司名称
