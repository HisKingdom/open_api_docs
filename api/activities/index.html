获取活动的AP

调用请求

HTTP请求方式: GET
https://api.convertlab.com/v1/activities?access_token={access_token}

参数说明

  • {access_token}用从“获取身份权限”节拿到的access_token替换

            "title": "string",
            "wechatImg": "string",

            "wechatSummary": "string",

            "type": "string",
            "wechatTitle": "string",
            "wechatRequireAuth": "string"
        },
        其他活动的信息...
    ]

获取绑定的微信公众号的API

调用请求

HTTP请求方式: GET
https://api.convertlab.com/v1/wechataccounts?access_token={access_token}

参数说明

  • {access_token}用从“获取身份权限”节拿到的access_token替换

返回参数

[
    {
        "id": 15,
        "name": "留学",
        "serviceTypeInfo": "2",
        "verifyTypeInfo": "0",
        "appId": "wxf4"
    },
    {
        "id": 15,
        "name": "升学",
        "serviceTypeInfo": "2",
        "verifyTypeInfo": "0",
        "appId": "wx9d"
    }
]

返回参数说明

  • id 绑定的微信公众号ID
    • name 微信公众号名称
    • serviceTypeInfo 服务类型信息
    • verifyTypeInfo 验证类型信息
    • appId 应用ID
